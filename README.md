# Duniter-ui

Duniter graphical interface. This is a developement package which is embedded in [Duniter software](https://git.duniter.org/nodes/typescript/duniter) on build phase.

## Installation

> Requires Nodejs v9
> Requires NVM
> Requires Yarn

```bash
nvm install 9
nvm use 9
yarn
```

## Run

```bash
node run.js direct_webstart
```
